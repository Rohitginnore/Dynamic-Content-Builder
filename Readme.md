# 🚀 Dynamic Content Builder (Drag & Drop Page Builder)

## 📌 Overview

This is a dynamic web application that allows users to create and customize their own page layout by adding text, images, and headings. Users can drag and drop elements to reorder them and save their layout using localStorage.

---

## ✨ Features

* ➕ Add Text, Heading, and Image blocks
* ✏️ Editable content (contentEditable)
* 🔄 Drag & Drop reordering (using SortableJS)
* 💾 Save layout using localStorage
* 🗑️ Delete individual blocks
* ❌ Delete All blocks
* 📱 Basic responsive design

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* SortableJS (for drag & drop functionality)
